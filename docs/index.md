# Car API

Bem-vindo à documentação oficial da **Car API**. Esta é uma API REST moderna e robusta desenvolvida com **FastAPI**, focada no gerenciamento de veículos e usuários.

## Objetivo
O objetivo deste projeto é fornecer uma plataforma escalável para o cadastro, consulta e gerenciamento de frotas de carros, garantindo segurança através de autenticação JWT e seguindo as melhores práticas de desenvolvimento Python.

## Funcionalidades Principais
- **Gestão de Usuários**: Cadastro, atualização e autenticação segura.
- **Gestão de Carros**: CRUD completo de veículos vinculados a proprietários e marcas.
- **Gestão de Marcas**: Controle de marcas de veículos.
- **Segurança**: Autenticação baseada em tokens JWT e controle de acesso (proprietário do recurso).
- **Documentação Automática**: Swagger UI e ReDoc integrados.
