# Contribuição

Obrigado por seu interesse em contribuir com a **Car API**! Toda ajuda é bem-vinda para tornar este projeto ainda melhor.

## Como Contribuir

### Reportando Bugs
Se você encontrar um problema, por favor abra uma **Issue** no GitHub descrevendo:
- O comportamento esperado.
- O comportamento observado.
- Passos para reproduzir o problema.
- Detalhes do seu ambiente (OS, versão do Python).

### Sugerindo Melhorias
Ideias para novas funcionalidades são bem-vindas! Abra uma Issue com a tag `enhancement` e explique por que a mudança seria útil.

### Enviando Pull Requests
1. Faça um **Fork** do projeto.
2. Crie uma branch para sua modificação (`git checkout -b feature/minha-melhoria`).
3. Faça suas alterações e adicione testes correspondentes.
4. Verifique se o código está formatado: `task format`.
5. Garanta que todos os testes passem: `pytest`.
6. Envie o **Push** para sua branch: `git push origin feature/minha-melhoria`.
7. Abra um **Pull Request** para a branch `main`.

## Código de Conduta
Mantenha a comunicação profissional e respeitosa em todas as interações do projeto. Valorizamos um ambiente colaborativo e amigável.
