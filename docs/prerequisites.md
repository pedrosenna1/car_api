# Pré-requisitos

Para rodar o projeto **Car API** localmente, você precisará garantir que seu ambiente possui as seguintes ferramentas instaladas:

## Sistema Operacional
- Recomendado: Linux (Ubuntu/Debian/Fedora) ou macOS.
- Windows: Funciona via WSL2 ou CMD/PowerShell (com ajustes menores).

## Ferramentas Necessárias
1. **Python 3.13 ou superior**:
   - Verifique com: `python --version` ou `python3 --version`.
2. **Poetry**:
   - Gerenciador de dependências e ambientes virtuais.
   - Instalação: [https://python-poetry.org/docs/#installation](https://python-poetry.org/docs/#installation)
3. **SQLite**:
   - Banco de dados padrão para desenvolvimento.
4. **Git**:
   - Para clonar o repositório.

## Conhecimentos Desejáveis
- Familiaridade com APIs REST.
- Conhecimento básico de Python e FastAPI.
- Noções de bancos de dados relacionais e SQL.
